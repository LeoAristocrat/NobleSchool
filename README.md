# NobleSchool

School website/front-end project built with React and Vite.

## Scope

This repository contains the website UI, animations, and page-level structure for the NobleSchool project.

## Tech stack

- React
- Vite
- TypeScript
- Framer Motion

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
